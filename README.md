# 邮轮产品智能助手 · Claude Code 插件

通过智能助手帮助客户快速找到合适的邮轮产品，轻松获取信息与建议。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-11-claude-plugin
/plugin install craftwave-skill-11@craftwave-skill-11-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **邮轮产品智能助手**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
